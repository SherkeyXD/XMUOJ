# 重复元素

| 题目链接 | [重复元素](http://xmuoj.com/problem/P013) |
| --- | --- |
| 可用语言 | Python3 |
| 时间限制 | 1000 MS |
| 内存限制 | 256 MB |
| IO 类型 | Standard IO |
| 难度 | 低 |

## 题目描述

<p><span style="color: rgb(55, 65, 81);">编写一个函数</span><span style="color: rgb(55, 65, 81);">，接受一个列表作为参数，返回列表中所有重复的元素。</span><br /></p>

## 输入

<p>输入一行列表里的元素，元素之间是空格隔开的</p>

## 输出

<p>列表中所有重复的元素</p>

## 样例

### 输入1

```
1 2 3 2 4 5 3
```

### 输出1

```
2 3
```
