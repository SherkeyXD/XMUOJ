| 时间限制 | 1000 MS |

## 题目描述

<p><span style="color: rgb(55, 65, 81);">实现一个函数</span><span style="color: rgb(55, 65, 81);">，接受一个字符串作为参数，返回字符串中只出现一次的字符，按照出现的顺序返回。</span><br /></p>

## 输入

<p>输入一个字符串</p>

## 输出

<p>输出<span style="color: rgb(55, 65, 81);">字符串中只出现一次的字符</span></p>

## 样例

### 输入1

```
abracadabra
```

### 输出1

```
c d
```
