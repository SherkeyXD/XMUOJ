| --- | --- |
| 可用语言 | C, C++, Java, Python3 |
| 时间限制 | 4000 MS |
| 内存限制 | 256 MB |
| IO 类型 | Standard IO |
| 难度 | 中 |

## 题目描述

<h2 style="margin-left: 0cm;"><img alt="image.png" src="/public/upload/e3296dc261.png" width="603" height="132" /><br /></h2>

## 输入

<p>输入共n+1行</p><p>第一行输入矩阵的总个数n[2,1000]</p><p>后n行分别输入矩阵的维数[1,100]</p>

## 输出

<p>最后一行输出少乘法次数<br /></p>

## 样例

### 输入1

```
6
30 35
35 15
15 5
5 10
10 20
20 25
```

### 输出1

```
15125
```
