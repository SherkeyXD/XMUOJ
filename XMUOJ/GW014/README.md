# 点和正方形的关系

| 题目链接 | [点和正方形的关系](http://xmuoj.com/problem/GW014) |
| --- | --- |
| 可用语言 | C, C++ |
| 时间限制 | 1000 MS |
| 内存限制 | 256 MB |
| IO 类型 | Standard IO |
| 难度 | 低 |

## 题目描述

<p>有一个正方形，四个角的坐标（x,y)分别是（1，-1），（1，1），（-1，-1），（-1，1），x是横轴，y是纵轴。写一个程序，判断一个给定的点是否在这个正方形内（包括正方形边界）。</p>

## 输入

<p>输入一行，包括两个整数x、y，以一个空格分开，表示坐标(x,y)。</p>

## 输出

<p>输出一行，如果点在正方形内，则输出yes，否则输出no。</p>

## 样例

### 输入1

```
1 1
```

### 输出1

```
yes
```

### 输入2

```
1 2
```

### 输出2

```
no
```
